# 🍕 Pizza Sales Analysis Dashboard

## Overview
End-to-end data analysis project for a pizza restaurant's 2015 sales data.
Built with Python, SQL Server, and Power BI.

## Tools & Technologies
- **Python** → Data cleaning & validation (Pandas)
- **SQL Server** → Database design, relationships & queries
- **Power BI** → Interactive dashboard

## Dataset
- 21,350 orders
- 48,620 order details
- 32 pizza types across 4 categories
- Full year 2015

## Key Insights
- 💰 Total Revenue: $817,860.05
- 📦 Total Orders: 21,350
- 🍕 Avg Order Value: $38.31
- 📅 Best Month: July
- 📆 Best Day: Friday
- ⏰ Peak Hours: 12 PM & 6 PM
- 🏆 Top Pizza: The Thai Chicken Pizza
